CommUnity

CommUnity is a web application designed to empower urban communities by fostering collective action and turning local activism into constructive correspondence with government officials. Originally conceived and developed at a hackathon at Rensselaer Polytechnic Institute (RPI), CommUnity features a Tailwind CSS frontend, a Firestore database for real-time data management, and a Flask API integrated with ChatGPT for generating impactful advocacy messages. The backend is hosted on AWS Lambda, ensuring scalability and reliability. This combination of modern technologies creates a streamlined platform for community-driven advocacy and meaningful change.
